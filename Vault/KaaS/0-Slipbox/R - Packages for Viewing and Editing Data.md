# R - Packages for Viewing and Editing Data

## Overview

### DataEditR

* [DataEditR]()

````R
install.packages("DataEditR")
library(DataEditR)
data_edit(iris)
````

### editData

* [editData]()

````R
remotes::install_github(“cardiomoon/editData”)
require(editData)
editData(iris)
````
